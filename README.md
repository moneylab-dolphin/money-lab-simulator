# マネーラボ 資産形成シミュレーター

note メンバーシップ「マネーラボ」読者向けの資産形成シミュレーター。
FIRE・サイドFIRE・老後の資産寿命・教育費・住宅購入vs賃貸をブラウザだけでシミュレーションできる静的Webアプリです。

## 構成

- `index.html` … 本体(これ1ページ)
- `css/style.css` … スタイル
- `js/engine.js` … 計算エンジン(シミュレーションロジック)
- `js/app.js` … UI・チャート描画
- `js/background.js` … 動的背景(WebGL星雲・オーロラ・恒星フレア+浮遊光粒子。
  タブ非表示時は描画停止、reduced-motion設定では静止画、WebGL非対応ではCSS光彩にフォールバック)

サーバー不要の静的サイトなので、GitHub Pages にそのまま置けます。

## 無料版と完全版(合言葉方式)

- 無料: 資産推移・FIRE可能年齢・サイドFIRE可能年齢・同年代比較(平均/中央値との比較まで)・逆算タブの操作
- 答えだけ限定(ブラー表示): 逆算の「必要積立額」、同年代比較の「上位◯%」と分布グラフ
- 完全版(合言葉入力で解放): 上記に加えて モンテカルロ成功確率 / 教育費の山 / 老後・取り崩し / 住宅購入vs賃貸 + 詳細パラメータ(配偶者・子ども・年金・退職金など)

「計算は終わっているのに答えの一歩手前で止まる」体験が、逆算・同年代比較タブの狙いです。

## 背景テーマ

右下の🎨ボタンで切替。localStorage に保存され再訪時も維持されます。

- 🌌 深宇宙(既定) / 🌠 星雲 / 🌊 水面 / 🏔 風景 / 🖌 金泥山水
- すべてWebGLシェーダーによるリアルタイム描画+浮遊粒子(テーマごとに胞子/泡/蛍/金砂に変化)
- 金泥山水は「紺紙金泥画(琳派の金屏風)」様式のオリジナル。手前の金の稜線は
  シミュレーションの資産カーブそのもので、スライダーを動かすと山の形が変わります
  (破綻プランなら山が崩れる)。奥山はシード生成で、余白ダブルクリックで生まれ変わります

## スクリーンセーバーモード

🎨メニューの「🌙 スクリーンセーバー」でUIを全部消し、背景だけを全画面で楽しめます。
戻るのはダブルクリック(ダブルタップ)かEsc。シングルクリックは「遊び」で、
クリックした場所にテーマごとの変化が生まれます:

- 深宇宙・星雲: 星が生まれる(衝撃波リング+光条+しばらく瞬いて消える)
- 水面: 波紋が広がり泡が立ちのぼる
- 風景: 蛍がぽっと湧いて散っていく
- 金泥山水: 金の墨がにじみ、金砂子が舞う

マウスカーソルは光る玉(テーマ色・尾を引く)に置き換わります。
スクリーンセーバー中はテーマごとの住人(衛星・彗星・流れ星・魚群・クジラ・鹿・くま・金の鳥)も現れます。

- 🌌 深宇宙: 人工衛星(点滅灯つき)・彗星・流れ星
- 🌠 星雲: 流れ星(高頻度)・彗星
- 🌊 水面: 魚群・クジラ
- 🏔 風景: 月光に照らされた鹿・くま(後光・接地影つき)
- 🖌 金泥山水: 金の鳥の群れ

動物・衛星・魚は「立体シルエット」方式(絵文字の形状に上下グラデーション陰影+
左上からのリムライト+接地影を合成)で、3Dモデル風の量感を出しています。
風景・金泥山水の月はシェーダーによるリアル描画(球体シェーディング+海+クレーター+周縁減光)。
マウスパララックスは全テーマで強め(視差で画面を「見回せる」感覚)+粒子は奥行き別に動きます。

## 結果画像のダウンロード(X投稿用)

「📸 結果画像を保存」ボタンで、FIRE可能年齢・必要資産・資産寿命・資産カーブ入りの
1200×675(Xカード比率)のPNGが保存されます。無料機能(拡散導線)です。

## 処方箋(自動診断)と記事リンクの設定

「💊 処方箋」タブは設定値から自動診断したコメントを表示します(無料)。
各診断カードに関連note記事へのリンクボタンを付けられます:

1. `js/app.js` 冒頭の `ARTICLES = { ... }` を開く
2. 該当キー(例: `emergency_fund`)に記事URLを貼る
3. URLが空のキーはリンクなしで診断文のみ表示されます

診断ルール: 資産枯渇 / 生活防衛資金不足 / 眠る現金 / インフレ0% / 強気利回り /
FIRE道筋なし / 教育費ピーク赤字 / 副収入提案(月5万で何年早まるか実計算) /
年金未設定 / NISA残枠 / 同年代上位20% / FIRE達成 / モンテカルロ成功確率(メンバー限定、
未解放者には解放CTAカードが出る)

## 新NISA消化率メーター

「資産推移」タブのチャート下に表示(無料)。左パネル「新NISAで使用済みの枠」を入力すると、
生涯投資枠1,800万円の消化率・残り枠・今の積立ペース(月30万円上限)で埋まる年齢・
65歳まで運用した場合の節税インパクト(含み益×20.315%)を試算します。

## PWA(ホーム画面アプリ化)

manifest.webmanifest + sw.js + icons/ でPWA対応済み。GitHub Pages(https)で公開すると、
スマホの「ホーム画面に追加」でアプリのように起動し、オフラインでも動作します。
Service Workerはネットワーク優先なので、通常は常に最新版が表示されます。
アイコンを変えたいときは icons/ 内のPNGを差し替えてください。

## タイトルの筆文字フォント

タイトル「資産形成シミュレーター」は、書道家・片岡佑二氏によるGoogle Fontsの筆文字
「Yuji Boku(佑字 墨)」を使用。`text=`パラメータでタイトルの文字だけをサブセット配信して
いるため軽量です。タイトル文言を変えるときは、index.html のフォント読み込みURLの
`text=`(URLエンコード済み文字列)にも新しい文字を追加してください。

## GitHub Pages での公開手順

1. GitHubで新しいリポジトリを作成(例: `money-lab-simulator`、Public)
2. このフォルダの中身をすべてアップロード(またはpush)
3. リポジトリの Settings → Pages → Source を「Deploy from a branch」、
   Branch を `main` / `(root)` にして Save
4. 数分後 `https://<ユーザー名>.github.io/money-lab-simulator/` で公開されます
5. そのURLをnote記事に貼ればOK

## シミュレーションの前提(読者向け説明にも使えます)

- 金額はすべて万円単位。年単位で100歳までシミュレーション
- 税金・社会保険料は「手取り」入力に織り込む簡略化
- 積立は在職中のみ。リタイア後は不足分を現金→リスク資産の順に取り崩し
- 年金・副収入・生活費はインフレ連動、給与は昇給率で成長
- 教育費は文部科学省「子供の学習費調査」等をもとにした概算(現在価値)をインフレ調整
- 大学は8分類: 国公立(4年)65万/年、私立文系105万/年、私立理系150万/年、
  国公立医歯薬(6年)65万/年、私立薬学部(6年)200万/年、私立歯学部(6年)530万/年、
  私立医学部(6年)は総額2,000〜5,000万円を選択(初期値3,500万)
- 利回りプリセット: 預金0.3% / 8資産分散4% / オルカン6% / S&P500 8%
  (過去実績目安より控えめの値。実績: S&P500 過去30年 約10%、全世界株 約7〜9%、いずれも円ベース)
- インフレ率プリセット: 過去30年平均0.5% / 日銀目標2% / 直近実績2.5%(2024年CPI総合は+2.7%・総務省)
- FIRE可能年齢 = 「その歳で完全リタイアしても100歳まで資産が尽きない最も早い年齢」
- 逆算タブ = 目標リタイア年齢に対し、100歳まで資産がもつ最小の毎月積立額を二分探索
  (収入を超える積立は反映されない=「収入的に不可能」なら達成困難と表示)
- 同年代比較 = 金融経済教育推進機構(J-FLEC)「家計の金融行動に関する世論調査」2024年の
  年代別・金融資産保有額分布(金融資産を保有していない世帯を含む/無回答除外で正規化)から
  階級内線形補間でパーセンタイルを推計。3,000万円以上の階級は3億円上限の対数補間
  - 出典: https://www.j-flec.go.jp/data/kakekin_2024/
- モンテカルロは 平均リターン±標準偏差の正規乱数 × 1,000試行
- 住宅比較は「金融資産 + 想定売却価値 − ローン残債」の純資産ベース
  (諸費用7%、維持費1%/年、建物残価は年1.5%逓減、ローン35年固定を仮定)
